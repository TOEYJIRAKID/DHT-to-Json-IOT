# ระบบเก็บข้อมูลอุณหภูมิและความชื้นลงในไฟล์ json
## Json Server Command
<code>json-server --watch data.json --host <input_ip> --port <input_port></code>

## โครงสร้างข้อมูล
```json
{
  "sensors": [
    {
      "humidity": 31,
      "temperature": 25.29999924,
      "timestamp": "07:53:48",
      "id": 1
    },
    {
      "humidity": 34,
      "temperature": 25.5,
      "timestamp": "08:19:03",
      "id": 2
    }
  ]
}
```
